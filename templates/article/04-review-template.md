## 三、分项评审

### 3.1 选题价值

**评分**：__/5

- [ ] 选题切中读者真实需求
- [ ] 选题有足够的差异化角度
- [ ] 选题的时效性合适

**评语**：________________

### 3.2 结构清晰度

**评分**：__/5

- [ ] 文章有清晰的逻辑线，从头到尾能串起来
- [ ] 每个段落/模块有明确的存在理由
- [ ] 段落之间过渡自然，没有突兀跳跃
- [ ] 信息密度均匀，没有头重脚轻或虎头蛇尾

**评语**：________________

### 3.3 内容深度

**评分**：__/5

- [ ] 论点有足够的论据支撑，不是空谈
- [ ] 案例/数据/引用真实可信
- [ ] 分析有深度，不停留在表面
- [ ] 有独到的见解或视角，不是常识搬运

**评语**：________________

### 3.4 表达质量

**评分**：__/5

- [ ] 语言简洁，没有冗余废话
- [ ] 用词准确，没有含糊或歧义
- [ ] 句式有变化，读起来不单调
- [ ] 专业术语有适当解释
- [ ] 没有语法/错别字问题

**评语**：________________

### 3.5 开头吸引力

**评分**：__/5

- [ ] 前 3 句话能让读者想继续看下去
- [ ] 开头与正文内容紧密相关，不是标题党
- [ ] 快速建立了"这篇文章和我有关"的感觉

**评语**：________________

### 3.6 实用性

**评分**：__/5

- [ ] 读者读完后知道下一步该做什么
- [ ] 建议/方法可操作，不是空泛的大道理
- [ ] 有具体的工具、步骤或框架可以直接套用

**评语**：________________
